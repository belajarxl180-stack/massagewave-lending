# **App Name**: MassageWave Landing

## Core Features:

- Hero Banner: Displays a prominent headline, tagline, product mockup, and a clear CTA button.
- Benefits Section: Highlights product benefits using a grid layout with icons.
- Product in Action: Showcases the product in action via embedded video and real-life photos.
- Customer Testimonials: Displays 4-6 customer reviews with profile pictures to build trust.
- Competitor Comparison: Presents a comparison table highlighting advantages over competitors.
- FAQ Section: Answers common questions (at least 6) to address potential customer concerns.
- WhatsApp Integration: Integrates a floating WhatsApp button for easy customer inquiries.
- Add-to-Cart Redirection: Adds products to cart and seamlessly redirects customers to the checkout link upon clicking 'Add to Cart'.

## Style Guidelines:

- Primary color: Deep calming blue (#3B82F6), reflecting relaxation and advanced tech.
- Background color: Very light desaturated blue (#F0F9FF).
- Accent color: Vibrant cyan (#06B6D4) for a modern touch and interactive elements.
- Body and headline font: 'Poppins' for a clean, geometric, contemporary style.
- Modern, rounded icons that visually represent the product features and benefits.
- Clean, conversion-optimized layout with large images and a modern UI following the style of Apple's product pages and tryjovi.com.
- Smooth scroll animations using Framer Motion to enhance user experience.
